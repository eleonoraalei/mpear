# Multi-bandpass Photometry for Exoplanet Atmosphere Reconnaissance (MPEAR) with the Habitable Worlds Observatory (HWO)

## Installation


1. Install pyEDITH following the official guide: https://pyedith.readthedocs.io/en/latest/installation.html
2. Install dockerized PSG in your local machine following the official guide: https://psg.gsfc.nasa.gov/helpapi.php#installation

Then, you are ready to install MPEAR!


>pip install -e .
